# Тесты

## Поиск фильмов

`python search.py`

Фаил search.py выполняет 7 тестов:
* Тест 1 - поиск по имени актера
* Тест 2 - поиск по имени режиссера
* Тест 3 - поиск по имени актера и режиссера
* Тест 4 - поиск по жанру
* Тест 5 - поиск по времени
* Тест 6 - поиск по имени режиссера и времени
* Тест 7 - поиск по имени с использованием регулярного выражения

## Статистика по актерам и режиссерам

`python statistics.py`

Фаил statistics.py выполняет 2 теста:
* Тест 1 - вывод фильмов для нескольких актеров
* Тест 2 - вывод фильмов для одного актера

## Поиск рекомендаций

`python recomendation.py`

Фаил recomendation.py проводит 1 тест и возвращает рекомендации для фильмов из файла films.txt
